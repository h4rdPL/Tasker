## Tasker - Task Management Platform

Welcome to **Tasker**, a task and project management platform! This application allows users to manage projects, create tasks, add comments, attach files, and collaborate with team members. Itâ€™s designed to streamline project workflows and enhance team productivity.

---

## Features

- **User Management**: Users can register, login, and manage their accounts securely.  

- **Projects & Members**: Create projects, invite members, and assign roles (Owner, Admin, Member, Viewer).  

- **Tasks**:  
  - Create tasks and assign them to users  
  - Add tags to organize tasks  
  - Set task priorities and status  

- **Comments & Mentions**: Communicate directly on tasks with comments and user mentions.  

- **Attachments**: Upload and manage files related to tasks.  

- **Project Invitations**: Invite new members to join projects via unique tokens.  

- **Responsive API Design**: Designed to support front-end clients like React or mobile apps seamlessly.  

---

## Usage

1. **Register & Login**: Create an account or login to access your projects and tasks.  
2. **Create Projects**: Add new projects and invite members.  
3. **Manage Tasks**: Create tasks within projects, assign members, set status/priority, and add tags.  
4. **Comments & Mentions**: Collaborate by commenting on tasks and mentioning team members.  
5. **Attachments**: Upload files to relevant tasks for easy access.  
6. **View Project Members**: Assign roles to control access and permissions.  

---

## Technologies Used

- **.NET**: Backend built with .NET Web API using Clean Architecture principles.  
- **Entity Framework Core**: ORM for database access and relationships.  
- **SQL Server**: Relational database for persisting tasks, projects, users, and related entities.  
- **REST API**: Structured endpoints for tasks, projects, comments, and users.  
- **Optional Frontend**: Can integrate with React.js, Angular, or mobile apps.
